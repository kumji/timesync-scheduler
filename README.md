# 🗓️ TimeSync Scheduler

**TimeSync Scheduler** is a specialized full-stack web application designed to streamline international class management by bridging the time gap between **Washington D.C. (EST/EDT)** and **South Korea (KST)**.

---

## Overview

In the world of global education, managing schedules across time zones is a common challenge.  
**TimeSync Scheduler** provides a centralized platform where users can easily register, track, and manage class schedules with **real-time timezone conversion**, ensuring that no session is ever missed due to time calculation errors.

The application is designed with clarity, efficiency, and usability in mind, making it ideal for educators and students participating in international learning environments.

---

## Key Features

### Dual-Time Dashboard
A high-visibility **weekly schedule grid** that displays class times based on **Washington D.C. time** while simultaneously showing the **corresponding Korean time**.

### Smart Timezone Engine
Automatically manages **time differences and Daylight Saving Time (DST)** adjustments between the United States and South Korea.

### Seamless Schedule Management
A simple and intuitive **modal interface** allows users to:
- Create schedules
- View class details
- Update existing entries
- Delete schedules

### Clean Dashboard UI
A professional and organized dashboard layout featuring:
- **Side navigation bar**
- **Mini-calendar for quick date reference**
- **Main weekly schedule grid**

## Purpose

**TimeSync Scheduler** simplifies global coordination by removing the complexity of timezone calculations.  
It enables educators and students to **focus on learning instead of worrying about time differences.**
